## ■ プロフィール
Webフロントエンドエンジニアとして、不動産申込・契約の電子化という複雑なドメインの新規開発から保守・運用まで約3年間従事。
2025年10月よりフロントエンドチームのリーダーを務めています。
「技術は事業成長の手段」という考えのもと、開発効率の向上、運用コストの削減、そしてチーム全体の納得感を醸成するコミュニケーションを重視しています。

---

## ■ 技術スタック

### Frontend
- **Language/Framework**: TypeScript, React, Next.js (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Jotai
- **Form Management**: React Hook Form
- **Validation Library**: Zod
- **Test Framework**: Vitest
- **Architecture**: カスタムフックによる責務分離, コンポーネント設計の最適化, 過度なメモ化の廃止によるレンダリング最適化

### Infrastructure / DevTools
- **Cloud**: AWS (App Runner, Amplify)
- **CI/CD**: GitHub Actions (ワークフロー最適化によるコスト削減)
- **Monitoring**: Sentry (不要エラーのフィルタリングによるコスト管理)
- **AI Integration**: Claude API を活用したプルリクエスト自動レビューフローの構築

#### 【主な取り組みと成果】

##### 1. 技術的な意思決定とアーキテクチャ設計
- **責務の明確化**: カスタムフックへのロジック抽出を徹底し、Propsの肥大化を抑制。長期運用に耐えうる保守性の高いコードベースを維持。
- **パフォーマンス改善**: 闇雲なメモ化（memo/useMemo）を廃止し、React本来の挙動を活かしたシンプルで高速な実装を推進。

##### 2. 開発効率化・コスト削減（Business Impact）
- **GitHub Actions最適化**: 実行条件を精査し、不要なワークフローを削減。CI/CDの実行コスト削減に直接貢献。
- **AI自動レビュー導入**: Claudeを連携させた自動レビューを構築。初歩的なミスの指摘を自動化し、人間によるレビューの質と速度を向上。
- **運用コスト管理**: Sentryのノイズを除外（ignore設定）し、超過料金の発生を抑制。

##### 3. チームマネジメント・リーダーシップ
- **透明性の高い意思決定**: 決定事項に対し「なぜそうするのか」という背景を丁寧に説明。メンバーの納得感と信頼関係を構築。
- **ポジティブなフィードバック**: コードレビューにおいて称賛文化を取り入れ、心理的安全性の高いチーム作りを実践。
- **クロスファンクショナルな連携**: デザイナーやバックエンドとの認識齟齬を早期解決するため、必要に応じて即座にミーティングを設定し、プロジェクトの停滞を防止。
